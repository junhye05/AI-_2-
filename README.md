## 💻 2. Python 코드 비교

### ❓ 질문
> Python으로 1부터 10까지 더하는 프로그램 만들어줘. (초보자용 주석 포함)

---

### 🤖 ChatGPT
```python
# 1부터 10까지의 합을 구하는 프로그램

total = 0  # 합계를 저장할 변수를 0으로 초기화

# 1부터 10까지 반복하면서 더하기
for i in range(1, 11):  # range(1, 11)은 1부터 10까지를 의미
    total += i  # total = total + i 와 같은 의미

# 결과 출력
print("1부터 10까지의 합:", total)
```

### 🧠 Gemini
```
# 합계를 저장할 변수를 0으로 초기화합니다.
total = 0

# range(1, 11)은 1부터 10까지의 숫자를 의미합니다.
# (마지막 숫자인 11은 포함되지 않아요!)
for i in range(1, 11):
    total = total + i  # 기존 합계에 현재 숫자(i)를 계속 더합니다.
    print(f"{i}를 더하는 중... 현재 합계: {total}")

# 최종 결과를 출력합니다.
print("-" * 20)
print(f"1부터 10까지의 총합은 {total}입니다.")
```

### 🎯 Claude
```
# 1부터 10까지의 합을 구하는 프로그램

total = 0  # 합계를 저장할 변수 (처음엔 0으로 시작)

# 1부터 10까지 반복 (range(1, 11)은 1~10을 의미)
for number in range(1, 11):
    total = total + number  # 현재 숫자를 합계에 더함
    print(f"{number}을 더했습니다 → 현재 합계: {total}")

print(f"\n최종 결과: 1부터 10까지의 합 = {total}")
```

## ✅ 비교 분석

- 👶 **가장 설명이 친절한 AI: Claude**  
  → 실행 과정과 결과까지 보여줘서 이해 쉬움  

- 🧹 **가장 깔끔한 코드: ChatGPT**  
  → 핵심 기능만 포함된 기본 구조  

- 📝 **주석이 가장 자세한 AI: Gemini**  
  → range 동작 등 세부 설명 포함

### 🤖 ChatGPT
- 공부법: “덜 힘들고 오래가는 방식” 찾기 (읽기/듣기/문제풀이 테스트)  
- 하기 싫은 일: 쪼개기 + 시작 장벽 낮추기 + 보상  
- 돈: 쉬운 방법은 없지만, IT 등 수요 높은 기술 + 결과 기반 구조  

---

### 🧠 Gemini
- 공부법: 인지 스타일 + 메타인지 + 다양한 학습법 실험  
- 하기 싫은 일: 5분 법칙 + 환경 설계 + 작업 흥분 유도

---

- 하기 싫은 일: 2분 시작 + 감정 분석
